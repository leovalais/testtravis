Test Travis CI

This repository is dedicated to experimentations with Travis CI. It is also a playground for experimentations with integrating the project html_of_wiki to a Travis CI. It is a documentation generator for GitHub Pages written in OCaml.

mymath

Contains some Python math functions. Please see the full list below:

fact
computes the factorial
inc
increments its parameter
negabs
negabs(x) = -abs(x)

test_mymath

Contains tests written for pytest of the functions described above.

Some tests are properties tests. Please see Haskell's QuickCheck for much more detail about that sort of tests. TODO: write a decorator @ptest to factorize the common parts
